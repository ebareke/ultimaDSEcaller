ultimaDSEcaller

Ultimate Differential Splicing Event caller — short- and long-read RNA-seq

From FASTQ to publication-ready differential-splicing results.

What it does

ultimaDSEcaller detects, quantifies, statistically tests and reports differential alternative-splicing events between groups of RNA-seq samples — from Illumina short reads and PacBio / Oxford Nanopore long reads alike. It is a single fast Rust binary, with a Nextflow pipeline and container images that cover the whole path from raw reads to interactive reports.

Highlights

18 event classes

Seven canonical plus eleven advanced splicing event types.

Short & long read

Junction PSI for Illumina; isoform reconstruction for PacBio/ONT.

Consensus statistics

Beta-binomial, GLM, GLMM and Fisher fused into one calibrated q-value.

Functional impact

Splice-site motifs, protein consequence and NMD prediction.

Rich reports

Volcano, MA, PCA, UMAP, heatmaps, sashimi, Sankey — one HTML file.

Reproducible

Docker, Apptainer and Nextflow run FASTQ → events in one command.

Install

# From source (Rust 1.85+)
git clone https://github.com/ebareke/ultimaDSEcaller.git
cd ultimaDSEcaller
cargo build --release --all-features

# Or build the container (bundles minimap2, STAR, HISAT2, samtools)
docker build -t ultimadsecaller:1.0.0 .

Run

From BAMs

ultimaDSEcaller run \
    --sample-sheet samples.tsv \
    --annotation gencode.v45.gtf.gz \
    --reference GRCh38.fa \
    --out results/ \
    --contrast group:treatment-control \
    --consensus stouffer

From FASTQ (Nextflow)

nextflow run nextflow/main.nf \
    --samplesheet samples.csv \
    --gtf annotation.gtf --fasta genome.fa \
    --contrast 'group:treatment-control' \
    -profile docker

Try the bundled example

cargo build --release
bash example/run_example.sh   # calls a synthetic exon-skipping event end-to-end
